Margens CSS

As CSS margin propriedades são usadas para criar o espaço em torno de elementos, fora de quaisquer fronteiras definidas.

Com CSS, você tem total controle sobre as margens. Existem propriedades de fixação da margem de cada lado de um elemento (em cima, direita e inferior, e à esquerda).

Margem - Sides individuais

CSS tem propriedades para especificar a margem de cada lado de um elemento:

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

Todas as propriedades de margem pode ter os seguintes valores:

  • auto - o navegador calcula a margem
  • comprimento - especifica uma margem em px, pt, cm, etc.
  • % - especifica uma margem em% da largura do elemento contendo
  • herdar - especifica que a margem devem ser herdado do elemento pai

Dica: Os valores negativos são permitidos.

O exemplo a seguir define margens diferentes para todos os quatro lados de um elemento <p>:

Exemplo 
                    p {
                    margin-top: 100px;
                    margin-bottom: 100px;
                    margin-right: 150px;
                    margin-left: 80px;
                    }

Margem - Propriedade Taquigrafia

Para encurtar o código, é possível especificar todas as propriedades de margem em uma propriedade.

A marginpropriedade é um atalho para as seguintes propriedades das margens individuais:

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

Então, aqui está como funciona:

Se a marginpropriedade tem quatro valores:

margem: 25px 50px 75px 100 p;
  • margem direita é 50px
  • margem inferior é de 75 pixels
  • margem esquerda é 100px
Exemplo 
                p {
                margin: 25px 50px 75px 100px;
                }

Se a margin propriedade tem três valores:

margem: 25 pixels de 50 pixels 75 pixels;
  • margem superior é de 25 pixels
  • margens direita e esquerda são 50px
  • margem inferior é de 75 pixels
Exemplo 
            p {
            margin: 25px 50px 75px;
            }

Se a margin propriedade tem dois valores:

margem: 25 pixels de 50 pixels;
  • margens superior e inferior têm 25 pixels
  • margens direita e esquerda são 50px
Exemplo 
            p {
            margin: 25px 50px;
            }

Se a margin propriedade tem um valor:

margem: 25 pixels;
  • todas as quatro margens são 25px
Exemplo 
                p {
                margin: 25px;
                }
