Definir altura e largura

A heighte width propriedades são utilizadas para definir a altura e a largura de um elemento.

A heighte width pode ser definido como auto (isto é padrão. Significa que o navegador calcula a altura e largura), ou ser especificado em valores de comprimento , como px, cm, etc., ou em percentagem (%) do bloco que contém.

Exemplo 
        div {
        height: 200px;
        width: 50%;
        background-color: powderblue;
        }
Exemplo 
        div {
        height: 100px;
        width: 500px;
        background-color: powderblue;
        }

Nota: A heighte widthpropriedades não incluem preenchimento, bordas, ou margens; eles definir a altura / largura da área no interior do
enchimento, fronteira, e a margem do elemento de!
