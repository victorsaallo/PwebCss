Cor de fundo

A background-colorpropriedade especifica a cor de fundo de um elemento.
A cor de uma página de fundo é definida assim:

Exemplo 
            body {
            background-color: lightblue;
            }

Com CSS, uma cor é mais frequentemente especificado por:

  • um nome de cor válido - como "vermelho"
  • um valor HEX - como "# FF0000"
  • um valor RGB - como "rgb (255,0,0)"

Olhe para valores de cor CSS para uma completa lista de possíveis valores de cor.
No exemplo abaixo, o <h1>, <p> e <div> elementos
têm diferentes cores de fundo:

Exemplo 
            h1 {
            background-color: green;
            }

            div {   
            background-color: lightblue;
            }

            p {
            background-color: yellow;
            }

Imagem de fundo

A background-imagepropriedade especifica uma imagem para usar como fundo de um elemento.
Por padrão, a imagem é repetida para que ele cobre o elemento inteiro.
A imagem de fundo para uma página pode ser definido assim:

Exemplo 
            body {
            background-image: url("paper.gif");
            }

Abaixo está um exemplo de uma má combinação de imagem de fundo de texto e. O texto é dificilmente legível:

Exemplo 
            body {
            background-image: url("bgdesert.jpg");
            }

Nota: Ao usar uma imagem de fundo, use uma imagem que não perturbe o texto.

Imagem de Fundo - Repetir horizontalmente ou verticalmente

Por padrão, a background-imagepropriedade repete uma imagem horizontalmente e verticalmente.
Algumas imagens deve ser repetido apenas horizontalmente ou verticalmente, ou eles vão olhar estranho, como este:

Exemplo 
            body {
            background-image: url("gradient_bg.png");
            }

Se a imagem acima é repetido apenas horizontalmente ( background-repeat: repeat-x;), o fundo vai parecer melhor:

Exemplo 
            body {
            background-image: url("gradient_bg.png");
            background-repeat: repeat-x;
            }

Dica: Para repetir uma imagem verticalmente, definirbackground-repeat: repeat-y;

Imagem de fundo - Definir posição e no-repeat

Mostrando a imagem de fundo apenas uma vez também é especificado pela background-repeatpropriedade:

Exemplo 
            body {
            background-image: url("img_tree.png");
            background-repeat: no-repeat;
            }

No exemplo acima, a imagem de fundo é mostrado no mesmo lugar como o texto. Queremos mudar a posição da imagem, para que não
perturbe o texto demasiado.

A posição da imagem é especificado pela background-positionpropriedade:

Exemplo 
            body {
            background-image: url("img_tree.png");
            background-repeat: no-repeat;
            background-position: right top;
            }

Imagem de Fundo - posição fixa

Para especificar que a imagem de fundo deve ser fixado (não vai rolar com o resto da página), use a background-
attachmentpropriedade:

Exemplo 
            body {
            background-image: url("img_tree.png");
            background-repeat: no-repeat;
            background-position: right top;
            background-attachment: fixed;
            }
